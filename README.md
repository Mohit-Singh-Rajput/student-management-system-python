# Student Management System (Python)

## Description
A CLI-based Python application to manage student records using file handling.
The system supports full CRUD operations with persistent storage.

## Features
- Add student records
- View all students
- Search student by roll number
- Update student details
- Delete student records

## Concepts Used
- Object-Oriented Programming (OOP)
- File Handling
- CRUD Operations
- Input Validation

## How to Run
1. Clone the repository
2. Run `main.py` using Python 3
